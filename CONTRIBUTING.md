# Contributing

empyrean is not currently accepting external code contributions. The project
is in an early phase and the authorial surface is deliberately narrow.

## What we welcome

- **Bug reports** — [file an issue](https://github.com/Empyrean-Dynamics/empyrean/issues)
- **Feature requests** — same tracker
- **Questions and discussion** — [GitHub Discussions](https://github.com/Empyrean-Dynamics/empyrean/discussions)
- **Validation findings** — if your results differ from ours, please tell us

## What we don't merge right now

- **Pull requests.** We won't merge externally-submitted code changes. Please
  open an issue to discuss any change you'd like to see before writing code.

This policy will open up as the project matures.
